# Project Portfolio Manager

A full-featured, single-file project portfolio manager built with vanilla HTML, CSS, and JavaScript.

## Features

- **Dashboard** — KPI cards: total projects, active, completed, average progress, tasks done
- **Projects** — Card grid view with status filters, search, progress bars, tech stack tags
- **Tasks** — Global task list with checkbox completion tracking per project
- **Statistics** — Bar charts for tech stack usage, donut chart for status distribution, progress histogram
- **GitHub Links** — Aggregated view of all project GitHub repository URLs
- **Add / Edit / Delete** projects with full form (name, description, status, progress, dates, tech stack, GitHub URL, priority)
- **Task management** — Add and complete tasks within each project's detail panel
- **Light & Dark mode** toggle
- **Persistent storage** — saves to localStorage
- **Responsive** — works on mobile and desktop

## Usage

Open `project-portfolio-manager.html` directly in any modern browser. No build step or server required.

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- [Lucide Icons](https://lucide.dev) via CDN
- [Inter + JetBrains Mono](https://fonts.google.com) via Google Fonts
- CSS custom properties (design tokens) with light/dark mode
- localStorage for persistence
